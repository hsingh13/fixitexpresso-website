MerryChef Error Codes

Find your code below. If the quick fix doesn't work, call us with the code displayed.

Try these yourself

E103 – Oven Overheating

Most Common

Electronics compartment overheated, usually a dirty air filter.

Try: Cool 30 min · clean air filter · clear ventilation.

E104 – Magnetron / Cavity Overheat

Microwave system or cavity overheated.

Try: Off immediately · clean air filter · cool 1 hour · restart.

E105 – Power Supply Issue

Voltage too high or too low.

Try: Check breaker · confirm dedicated circuit · off 5 min · restart.

E107 – Communication Error

Internal control boards aren't talking.

Try: Power off 5 min · restart.

E115 – Convection Fan Failure

Main cooking fan isn't running right.

Try: Restart · listen for the fan on startup.

E117 – Magnetron Overheat

Microwave system overheated and needs to cool.

Try: Cool fully · clean both filters · clear ventilation.

Call us for these

These need a technician. Note the code and call.

E101 – Microwave System Failure

Microwave system failed to start.

E102 – Heater Current Fault

Heating element drawing wrong current.

E106 – Cavity Temp Too High

Oven temperature exceeded safe range.

E108 – Personality Module / SD Card Error

Can't read configuration module or SD card.

E109 – Personality Module Failure

Personality module missing or invalid.

E110 – Firmware Version Conflict

Control boards have incompatible firmware.

E111 – Cavity Temp Sensor Error

Can't read cavity temperature sensor.

E113 – Microwave Safety Fault

Magnetron stayed on when it should have turned off. Stop using oven.

E116 – Heater Failure

Heating element failed to turn on.
